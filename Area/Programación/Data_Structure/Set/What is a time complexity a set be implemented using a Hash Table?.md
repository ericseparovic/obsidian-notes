| **Operation**            | **Average Time Complexity (O)** | **Worst-case Time Complexity (O)** |
|--------------------------|---------------------------------|------------------------------------|
| **Insert**               | O(1)                            | O(n)                               |
| **Delete**               | O(1)                            | O(n)                               |
| **Search (Contains)**    | O(1)                            | O(n)                               |
| **Union**                | O(N₁ + N₂)                     | O(N₁ + N₂)                        |
| **Intersection**         | O(N₁ + N₂)                     | O(N₁ + N₂)                        |
| **Difference**           | O(N₁ + N₂)                     | O(N₁ + N₂)                        |
